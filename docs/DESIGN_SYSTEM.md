# VibeTag: UI/UX Design System

## Visual Identity
- **Primary Brand Color:** `#FF4E6B` (Apple Music Pink/Red).
- **Backgrounds:**
  - Light Mode: `#FFFBFB` (Off-white/Blush).
  - Dark Mode: Midnight Blue.
- **Style:** "Soft Glass" (Native iOS Glassmorphism).
- **Typography:** SF Pro Rounded.

## Key Components

1. **Header Area:**
   - **Top Left:** "Pill" button acting as Provider Selector (Apple Music Logo + Text).
   - **Top Right:** User Avatar.

2. **AI Search Bar:**
   - **Style:** Must mimic "Apple Intelligence" (iOS 18) aesthetics.
   - **Effect:** Animated multi-color glowing stroke/gradient (Cyan/Purple/Pink) indicating active intelligence.

3. **Song Cells:**
   - **Layout:** Minimalist. Artwork (Left) + Title/Artist (Middle) + Tag Chips (Bottom).
   - **Constraint:** NO "three dots" menu. Keep the interface clean.

4. **Tagging Interface:**
   - **Presentation:** Modal Sheet.
   - **Layout:** Grid of "Pill" style tags.
   - **Colors:** Pastel palette for the tags.
